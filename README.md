# laravel-starter-api

This is a REST API for authentication. 

Built with laravel.

## Available routes

<ul>
<li>localhost/api/public/login </li>
<li>localhost/api/public/loginCheck </li>
<li>localhost/api/public/register </li>
<li>localhost/api/private/users  </li>
</ul>

## Installation

npm install

## Usage

php artisan serve
